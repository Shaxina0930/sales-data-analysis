# E-Commerce Sales Data Analysis
Analyze 2019 sales data to uncover trends in products, cities, and timing (185k+ records).

### Key Insights
- Total Revenue: ~$34M (assuming full data).
- Top Product: Macbook Pro Laptop (~$8M in sales).
- Best Month: December (holiday surge).
- Peak Cities: San Francisco, New York.
- Optimal Ad Times: Evenings (6-8 PM).

### Visualizations
- Monthly sales bar chart.
- Hourly trends line plot.

### Tech Stack
Python, Pandas, Matplotlib/Seaborn.
